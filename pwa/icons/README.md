# PWA Icons

Icons are generated by `generate_icons.py`. The four PNGs in this directory
are committed pre-built and used as the v1 defaults:

- `icon-192.png` — 192×192, browser tab + Android home screen
- `icon-512.png` — 512×512, Android splash + larger contexts
- `icon-maskable-512.png` — 512×512, Android adaptive icon (safe-zone aware)
- `icon-favicon.png` — 32×32, browser favicon

## Regenerating

```bash
python3 generate_icons.py                                 # default: "O" in blue/purple gradient
python3 generate_icons.py --letter J                      # for a customer named Joe's
python3 generate_icons.py --color "#ff6b35"               # change primary color
python3 generate_icons.py --color "#ff6b35" --color2 "#ffd166"  # change gradient end color
```

Requires Pillow:

```bash
pip3 install --break-system-packages pillow
```

## Per-customer branding (Phase 2)

In Phase 2 each customer will get their own icon during onboarding. The
onboarding wizard will run `generate_icons.py --letter X --color Y` based on
the business name and brand colors they provide.

For Phase 1, every customer uses the default Orbi icon.
