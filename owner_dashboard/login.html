📱 Add Orby to your phone

On iPhone — use Chrome (not Safari)

Important: Safari blocks Orby's voice when installed. Chrome works fully.
  1. If you don't have Chrome, get it from the App Store (free).
  2. Open Chrome and go to this same Orby URL.
  3. Sign in.
  4. Tap the ⋯ menu (three dots, bottom-right of Chrome).
  5. Scroll down, tap Add to Home Screen.
  6. Tap Add. Done — Orby icon on home screen.

On Android — Chrome or Edge

  1. Open Chrome (or Edge / Brave / Samsung Internet).
  2. Go to this Orby URL and sign in.
  3. Chrome shows a banner at the bottom — "Install Orby" — tap it.
  4. If you miss the banner: tap the ⋮ menu top-right → Install app.
  5. Done — Orby icon on home screen, opens like a real app.

On desktop (Mac / Windows / Linux)

  1. Open Chrome or Edge.
  2. Sign in to Orby.
  3. Look for the install icon in the address bar (looks like a screen with a down-arrow), OR ⋮ menu → Install Orby.
  4. Click Install. Orby gets a real app window + dock/taskbar icon.

After installing, sign in with your username + password. Each staff member uses their own credentials to get into their own per-user data.
